# yourhealth
Лабораторна робота №2
Студентки Шості Марії, групи ЗК-33

  YourHealth+ – це веб-платформа, яка виступає інтерактивною бібліотекою
дієт та харчових планів, орієнтованих насамперед на людей із харчовими
обмеженнями та особливими потребами. Сервіс допомагає користувачам
підбирати раціон у випадках алергій, непереносимостей (глютен, лактоза
тощо), захворювань шлунково-кишкового тракту (гастрит, виразкова
хвороба, панкреатит) та інших станів, де важливе правильне харчування.

Структура:
/css – файли стилів  
/images – зображення та медіа  
/js – JavaScript файли  
index.html – головна сторінка  
diets.html – каталог дієт  
recipes.html – рецепти  
communities.html – спільноти  
profile.html – профіль  
about.html – інформація про платформу  
contacts.html – контакти
